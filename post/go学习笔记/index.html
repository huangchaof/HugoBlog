package main

import (
	"fmt"
	"strconv"
)

func main() {
	var str string = "dsahj我"
	fmt.Printf(str)

	var a [3]int             // array of 3 integers
	fmt.Println(a[0])        // print the first element
	fmt.Println(a[len(a)-1]) // print the last element, a[2]

	// Print the indices and elements.
	for i, v := range a {
		fmt.Printf("%d %d\n", i, v)
	}

	// Print the elements only.
	for _, v := range a {
		fmt.Printf("%d\n", v)
	}

	i, err := strconv.Atoi(str)
	print(i)
	println(err)
}

go的设计原则

运行效率，编译效率，部署效率，开发效率

go是静态编译型语言，语法类似c，却又带有GC，不要开发者担心内存管理，go源代码编译后会生成一个二进制文件，部署时不需要像java要带jre，或者python带解释器，go的runtime已经编译进了这个二进制文件

goformt对go源代码进行格式化，且这个命令行工具没有自定义参数，所以go的代码风格统一

go没有虚拟机，只有运行时，这一点和java不同

从Go语言的设计原则分析其适用性 - 知乎 (zhihu.com)

万字长文深入浅出 Golang Runtime - 知乎 (zhihu.com)

go提供了runtime 包，支持修改运行时

runtime package - runtime - pkg.go.dev

go的协程是原生的，支持并发，这适合高并发应用

go的另一特点就是简化，没有宏、继承、函数重载等特性

go可以看做是更好的c或者更好的python

c需要自己管理内存，编译复杂，没有包管理机制，python是动态类型，GIL存在导致不能有效使用多线程，且python的执行效率不够高，分发时体积大
